Writing Circle on GitHub
A mobile app written in React Native for my senior project in University.
A somewhat ambitious project where users can share their creative writing and discover other writers' work.
The scope ended up being larger than what I could accomplish at the time, essentially being a social media platform. I was able to implement a working profile system that allowed users to share their work and view a universal list of stories obtained through a Firebase database. My troubles included:
Realizing I didn't understand Typescript, and then manually changing all of my files to Javascript in order to save time.
Starting the project over because I started with a navigation template that I didn't understand to save time. I ended up starting from scratch, and learning a lot about how navigation works.
And finally, accidentally implementing an infinite loop on a database read, which used up my month's quota of database operations in about 2 seconds, forcing me to create a new database and carefully fix the loop without the ability to run the code.
Despite these trials and not being able to fully finish the project, I had learned a lot and was praised for my impressive, albeit rather stupid, ambition for the project.
